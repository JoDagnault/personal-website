## Qu'est-ce que c'est

Un site web statique construit avec Astro pour présenter mes projets.

## Pourquoi je l'ai construit

Je voulais un site simple, entièrement sous mon contrôle, pour présenter mon
travail. C'était aussi un projet pour pratiquer mes compétences frontend.

## Stack et décisions

Je voulais que le site soit rapide et simple.
Je voulais aussi rester proche du HTML, CSS et JavaScript natifs pour
pratiquer mes fondamentaux frontend.

Chaque projet est rédigé en MDX, ce qui sépare le contenu de la mise en page.
Le routage bilingue est intégré directement dans l'URL.

Hébergé sur Cloudflare Pages.

## What it is

A static website built with Astro to showcase my projects.

## Why I built it

I wanted a simple website that I fully control to showcase my work.
It also gave me a project to practice my frontend skills.

## Stack & decisions

I wanted the website to be fast and simple.
I also wanted to stay close to native HTML, CSS and JavaScript to practice my frontend fundamentals.

Each project is authored as an MDX file, so content and layout stay separate.
The bilingual routing is built directly into the URL.

Hosted on Cloudflare Pages.
